## 技术架构

```
云端服务器集群
    ↓ WebSocket / HTTP API
云手机实例 (Android 10+)
    ↓ 独立公网IP
目标平台 (TikTok/FB/IG等)
```

- 基于 AOSP 定制 Android 系统
- 独立网络命名空间，每设备独立IP
- RESTful API + WebSocket 实时控制
- 支持 ADB 远程调试、屏幕串流
